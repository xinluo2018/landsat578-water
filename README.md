# **landsat578-water**
Deep learning-based surface water mapping by using Landsat-5, -7 and -8 images.
### -- **Objectives**
- 100 landsat images labeling.  
- High-accurate surface water mapping (oa/miou >> 90%) by using deep ConvNet models.
